### 디자인 패턴

------

##### 유형 분류

1. ##### Cretional Pattern

   > 객체 생성에 관련
   >
   > 객체 생성 시, 유연성 높이고 코드의 유지보수에 도움

2. ##### Structural Pattern

   > 프로그램 구조 관련
   >
   > 자료구조 or 인터페이스 등 프로그램 구조 설계에 활용

3. ##### Behavioral Pattern

   > 반복적으로 사용되는 객체들의 상호작용 패턴화
